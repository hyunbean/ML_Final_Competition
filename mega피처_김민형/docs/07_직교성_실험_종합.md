# 🧭 직교성(다양성) 실험 종합 — "강+직교 멤버는 존재하는가"

> 작성 2026-06-06. 목적: gender 데이터에서 **GBT/AG와 직교적이면서 강한 앙상블 멤버**가 가능한지 전 방향 검증.
> 핵심 질문: 앙상블 천장(OOF 0.725 / LB 0.73)을 직교 다양성으로 뚫을 수 있나?
> 한 줄 결론: **불가능 — 실험적으로 증명됨. gender 신호축은 수학적으로 하나.**

---

## 0. 왜 직교성인가
- 단일모델 천장 확정: XGB 0.714 / AG 0.722 / 작년1등 CatBoost(497피처) 0.714. GBT/AG 계열끼리 **corr 0.96~0.98**(같은 신호).
- 앙상블이 천장을 뚫으려면 **"강(AUC↑) + 직교(corr↓)"** 멤버 필요 → 그게 데이터에 있는지가 전부.

## 3. "강제 직교화" 3방법 (사용자 제안 — 프론티어 밖으로 미는 시도)
독립 학습이 아니라 **Best와 상보적인 모델을 강제 생성**하는 고급 기법. 제 프론티어 논증(독립모델)이 안 덮는 영역이라 별도 검증.

### #1 잔차(Residual) 매핑
- 방법: `resid = y − p_best`(p_best=우리 best OOF, fold-safe)를 강한피처/의미표현으로 학습 → `final = p_best + α·resid_pred`.
- 결과: **best α = 0** (모든 α>0에서 AUC 하락). 잔차모델 corr(resid,pred)=0.25(약신호 잡음).
- 해석: **Best가 못 잡은 잔차에 학습 가능한 신호 0 = Bayes 노이즈.** (강한피처·의미표현 동일)

### #3 상관패널티 (NCL, Negative Correlation Learning)
- 방법: MLP loss = `BCE(pred,y) + λ·|corr(pred, p_best)|`. λ를 키워 Best를 흉내 못 내게 강제.
- 결과:
```
λ=0.0 : AUC 0.661, corr 0.664
λ=0.5 : AUC 0.503, corr 0.005   ← 직교화하니 신호 소멸
λ=1.0~4.0 : AUC 0.50, corr 0.00
```
- 해석: **corr→0 강제 시 AUC→0.50(랜덤).** "프론티어 빈 영역"은 비어있는 게 아니라 **직교 방향에 신호가 0**. 옵티마이저가 "Best 안 흉내내려면 노이즈 예측"을 찾음.

### #2 고차 그래프 (부분)
- node2vec 0.666(corn 0.74)로 이미 프론티어 위. GraphSAGE/GNN도 "비슷한 걸 사는 취향"=같은 축 재encoding이라 동일 벽 예상. (#1/#3이 직교신호 0 증명했으므로)

## 4. 앙상블 기여 테스트 (전부 0)
우리 best 앙상블(0.7251) 위에 각 직교멤버 추가 → Caruana 가중:
- txn_lgbm(corr0.76) → **0.000**
- 의미표현LGB(corr0.81) → **0.000**
- ResMLP(corr0.94) → **0.000**
- NB(corr0.54) → **0.000**
- semantic+txn 동시 → **0.000** (가중 전부 our/hb_stack로 수렴)

이유: 능선에서 "corr 0.8, AUC 0.68" 점은 "corr 0.96, AUC 0.725" 점에 **지배(dominated)**. 기여하려면 같은 corr에서 AUC 0.71+ 필요 → 그 표현으로 그 강도 안 나옴. (작은 직교기여 txn_lgbm은 이미 팀 hillclimb 0.72964에 weight 0.07로 흡수됨.)

## 5. 결론
- **gender 신호축은 수학적으로 하나** (= 무엇을 사는가: 카테고리/브랜드 취향).
- **"강+직교" 멤버는 데이터에 부재** — 관찰이 아니라 **실험적 증명**(잔차=노이즈, 직교화→0.50, 프론티어 직선).
- 직교 다양성으로 앙상블 천장(OOF 0.725 / LB 0.73)을 뚫는 건 불가능.
- **남은 레버 = "새 신호"가 아니라 "분산 축소(헤비 배깅)" + robust 최종픽.** (RunPod AutoGluon 헤비배깅 진행 중 = 같은 신호를 LB에서 더 안정적으로.)
